## Getting Started

1. **Clone the repository**:

   ```bash
   git clone https://github.com/outfitter-dev/monorepo
   cd monorepo
   ```

2. **Install dependencies**:

   ```bash
   bun install
   ```

3. **Build all packages**:

   ```bash
   bun run build
   ```

4. **Run tests**:

   ```bash
   bun test
   ```

See the [Development Guide](development.md) for detailed instructions.
